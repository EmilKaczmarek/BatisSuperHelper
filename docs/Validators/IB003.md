## Description
Map files should have Build Action of "Embedded Resource".

## Default severity
Warning

## Fix
Change file Build Action to "Embedded Resource".

## Extension fix/lightbulb
No fix is provided.

## Future plans
Add ability to change Build Action to embedded, from code window.
